# Krese Landing Page

The official landing page for Krese Goal Tracker.

## Deployment

This landing page is deployed to **Cloudflare Pages** at `startkrese.com` and `www.startkrese.com`.

### How to Deploy

1. Push changes to the `main` branch
2. Cloudflare Pages will automatically build and deploy

### Local Testing

Open `index.html` in your browser to preview locally.

```bash
# Simple HTTP server (Python 3)
python -m http.server 8000

# Or with Node.js
npx http-server
```

Then visit `http://localhost:8000`
